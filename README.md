# FaceRecognition
A webapp that recognises faces in the pictures uploaded by the users, and maintains a leaderboard with the highest number of points based on correct facial recognitions!  

## Local Usage 
The application currently works in development mode by simply booting up a Node.js server. After cloning the repository, follow these steps to get it running yourself -:

Clone the repo and install the dependencies.
```
$ git clone https://github.com/uddhav99/FaceRecognition.git
$ cd FaceRecognition
```
```
npm install 
```
After all the dependencies are installed you can start the server
```
$ npm run start:dev
```

## Technologies
- Front-end: React.js, JavaScript, HTML5/CSS3
- Back-end: Node.js, Express, SQL
